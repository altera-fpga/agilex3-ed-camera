# ISP-based Camera Solution System Example Designs for Agilex™ 3 Devices - Repository

## Overview

This repository contains projects demonstrating how to create and build
ISP-based Camera Solution Example Designs, using Altera®'s [Video and Vision Processing (VVP) Suite IP cores](https://www.altera.com/products/ip/po-3150/video-and-vision-processing-suite),
and targetting Altera® Agilex™ 3 devices.

The projects are stored in individual folders.
